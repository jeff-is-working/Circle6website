SBOM Manager

Capabilities

What You Get

Everything an enterprise needs for open-source license compliance, from automated ingestion to legal review board submission.

SBOM Ingestion

CycloneDX and SPDX parsing from CI/CD pipelines. Component deduplication across projects and product versions. Push from any SCA tool.

Classification Engine

6-step automated pipeline: exclusion filters, pre-approved lists, historical approvals, version-bump detection, and license policy evaluation.

Review Workflow

Batch approve and reject with metadata editing. Full audit trail with who approved what, when, and why. Link to external ticketing systems.

Multi-Tenant

Full organization isolation with OIDC single sign-on and role-based access control. Viewer, submitter, approver, and admin roles.

Plugin System

Integrate ticketing systems like ServiceNow, notification channels, and custom approval workflows through a typed plugin API.

API-First

Complete REST API with token authentication for CI/CD integration. Helm charts for Kubernetes deployment. Prometheus metrics built in.
